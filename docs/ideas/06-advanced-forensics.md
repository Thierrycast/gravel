# Ideia: Forense Financeira e Análise Probabilística

Aproveitar o poder de processamento local para análises que seriam custosas ou invasivas em sistemas de nuvem.



## 2. Detecção de Anomalias Estatísticas
- **Benford's Law Check:** Analisar a distribuição dos dígitos iniciais das transações para identificar inconsistências nos dados importados.
- **Hidden Subscription Detector:** Identificar transações que ocorrem com periodicidade exata (ex: a cada 30 ou 31 dias), mas com valores ligeiramente diferentes, sinalizando assinaturas com taxas variáveis.
