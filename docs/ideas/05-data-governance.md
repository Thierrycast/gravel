# Ideia: Soberania e Contabilidade

Ferramentas para usuários avançados e conformidade.

## 1. Métodos Contábeis (Crypto)
- **Custo Base:** Permitir a escolha do método de cálculo para lucro/prejuízo em cripto: Preço Médio, FIFO ou LIFO.
- **Normalização de Sinais:** Configurar como o sistema interpreta sinais de provedores específicos (como o Pluggy faz com cartões de crédito).

## 2. Danger Zone
- **Unlink Provider:** Deletar todos os dados brutos e projetados de um provedor específico (ex: "Resetar apenas Binance").
- **Factory Reset:** Limpar todo o banco de dados e arquivos de configuração.
