# Integração Binance

O Gravel sincroniza ativos cripto, preços, trades e saldos via Binance quando as credenciais estão configuradas.

```env
BINANCE_API_KEY=
BINANCE_API_SECRET=
```

As telas `/crypto` e `/portfolio` usam esses dados para composição patrimonial. O patrimônio líquido consolida fiat, investimentos, cripto e passivos/faturas.
